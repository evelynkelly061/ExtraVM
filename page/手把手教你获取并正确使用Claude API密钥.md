1. **添加API密钥到请求头**  
   在发送API请求时，将API密钥添加到Headers中的`x-api-key`字段。  
2. **设置请求参数**  
   根据需求配置请求Body，确保包含必要的数据和参数。  
3. **发送请求并获取响应**  
   调用API后等待返回结果，处理Claude生成的回复。
